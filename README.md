# 反诈十六型互动测评

这是一个纯静态网页，可直接打开 `index.html` 使用，也可以部署到 GitHub Pages、学院网站、公众号菜单链接或任意静态服务器。

## 内容结构

- `12` 道案例题：覆盖刷单返利、虚假理财、冒充客服、校园贷、养老诈骗等高发类型。
- `4` 个测评维度：核验倾向、识别方式、沟通重点、行动风格。
- `16` 种反诈人格：答题后生成 MBTI 风格代码、中文类型名、优势打法、盲点提醒、适合岗位和分享文案。
- 案例速查卡：可在线下宣讲时展开讲解。
- 今日反诈行动墙：本机本页保存，适合线下互动展示。

所有内容都在 `index.html` 内：

- 16 型名称和文案：搜索 `const typeMeta = {`。
- 案例速查卡：搜索 `const caseCards = [`。

页面无外部依赖，不需要 npm、Node 或后端服务。
